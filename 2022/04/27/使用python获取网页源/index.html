XPath

  • XPath（XML Path）是一种查询语言，它能在XML（Extensible MarkupLanguage，可扩展标记语言）和HTML的树状结构中寻找结点。
  • XPath语法：

    获取文本：//标签1[@属性1=”属性值1”]/标签2[@属性2=”属性值2”]/…/text()
    获取属性值：//标签1[@属性1=”属性值1”]/标签2[@属性2=”属性值2”]/.../@属性n

  • 从拥有标志属性值的标签开始定位
  • 特殊情况：

    属性以某些字符串开头：//标签[starts-with(@属性名,”相同的开头部分”)]
    属性包含某些字符，”starts-with”换成contains

    XPath使用示例

    使用string(.)就可以把数据获取完整，使用普通方法不能获取到内部标签的数据
    selector = lxml.html.fromstring(html3)
data = selector.xpath('//div[@id="test3"]')[0]
info = data.xpath('string(.)')

    Beautiful Soup4

  • Beautiful Soup4（BS4）是Python的一个第三方库，用来从HTML和XML中提取数据。Beautiful Soup4在某些方面比XPath易懂，但是不如XPath简洁，而且由于它是使用Python开发的，因此速度比XPath慢。
  • 使用Beautiful Soup4提取HTML内容，一般要经过以下两步。（1）处理源代码生成BeautifulSoup对象。（2）使用find_all()或者find()来查找内容。
  • 首先使用解析器解析源代码，soup = BeautifulSoup(source,’lxml’)
  • 通过find或find_all方法定位，find(class=”test”)
  • 返回的对象类型通过.string属性读出标签中的文字信息

    find()与find_all()的不同点

  • find_all()返回的是BeautifulSoup Tag对象组成的列表，如果没有找到任何满足要求的标签，就会返回空列表。
  • find()返回的是一个BeautifulSoup Tag对象，如果有多个符合条件的HTML标签，则返回第1个对象，如果找不到就会返回None。find_all()与find()的参数完全相同

    find_all(name,attrs,recursive,text,**kwargs)
