1.数组

array.array对象和数组

array对象是包含相同基本数据类型的列表，操作与list对象基本一致，区别是创建时必须指定其元素类型typecode，且元素只能为该类型，否则将导致TypeError。array对象支持包括索引访问，切片操作，连接操作，重复操作等。array是可变对象，支持删除操作，改变其元素的值。

import array
a = array.array('b',(1,2,3,4,5))
print(a[1])#2
a[1]=22
print(a[1:])#array('b', [22, 3, 4, 5])

2.栈和队列

栈

栈的实现可使用列表，list.append()入栈，list.pop()出栈

deque对象

collections.deque双端队列支持从任意一端增加和删除元素。
deque([iterable[,maxlen]])
deque作为队列，入队：deque.append(),出队：deque.popleft()。

集合

set中为任意可hash对象，元素不可重复，且无序，存储依据对象的hash码。所有内置不可变对象（bool,int,float,complex,str,tuple,frozenset）等都是可hash对象。所有内置可变对象（list,dic,set）都是非hash对象。s={元素1，元素2…}

3.字典

字典的键只能使用不可变对象，但字典的值可以使用不可变或可变对象。
{“键1”:”值1”…},d.keys()返回字典的键key列表，d.values()返回值列表，d.items()返回d(key,values)对的列表。

defaultdict对象

可以使用构造函数参数指定键值对中值类型，键不存在时返回默认值。
